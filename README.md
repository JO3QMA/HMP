HMP
====

設定可能な小さなバックアップツール。
名前は人類は衰退しましたのヒト・モニュメント計画の英訳、`Human Monument Project`の頭文字から来ています。

## Description
簡単に**ラフな**バックアップを行いたいと思い、GSuiteのGoogle Driveとローカルでそれぞれバックアップを保持するツールです。
Google Driveは無制限なのでフルバックアップのみ対応予定です。
